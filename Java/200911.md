```
*.inc 파일을 웹서버가 기본설정일 경우 특별히 서버에서 실행하지 않고 일반 TEXT 취급
*.inc는 단독으로 실행되는 경우가 없으므로 웹상에서 그냥 TEXT형식으로 보이게 된다.
*.inc파일에 중요한 정보가 있다면 노출될 위험이 있으므로 중요한 정보는 기입하지 않는 것이 좋다.

*.inc파일은 웹프로그래밍을 할때 include를 위한 확장자이다.
간단하게 index.php나 다른페이지에서 항상 사용되는 함수나 HTML코드들을 담아두고 include해서 사용

```
